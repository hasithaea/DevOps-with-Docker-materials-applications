# DevOps with Docker — Course Material Applications

Course provided applications from the **DevOps with Docker** course conducted 
by the **University of Helsinki**.

> Original projects are provided by University of Helsinki.  
> Dockerfiles and container configurations are my own work.

## Course Info
- **Course:** DevOps with Docker, Spring 2026
- **Institution:** University of Helsinki
- **Platform:** [mooc.fi](https://www.mooc.fi/en/)

## About
This repository contains real-world applications provided by the course. 
Each application is containerized using Docker as part of the course exercises.
